
# Formulário de Pedidos da Cafeteria

Este pacote contém:
- index.html: Página para realizar pedidos de combos e produtos.
- visualizar.html: Página para visualizar os pedidos salvos na planilha do Google Sheets.
- README.txt: Instruções de uso.

## Como usar

1. Suba os arquivos `index.html` e `visualizar.html` em um serviço como GitHub Pages ou Netlify.
2. Certifique-se de que o script do Google Apps esteja publicado e com permissões públicas.
3. Edite o campo `window.open()` no botão "Mostrar" para apontar para o link da sua planilha se necessário.
4. A senha atual para salvar pedidos é: `cafeteria123`

Bom uso! ☕
